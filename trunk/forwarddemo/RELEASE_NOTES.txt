This is a collection of demo applications for ForwardFW.
The demos will help you to understand the system and shows you the usage of the
framework.

A demo for the caching system was added.

Requirements:

- ForwardFW 0.0.11

- PHP 5.4.0 or newer

- You'll need the FirePHPCore extension, which you can find on http://www.firephp.org.

- MDB2 2.4.0 or newer

- You can use Smarty Template Engine (Version 3.1 or newer), which you can find on
http://www.smarty.net or the Twig Template Engine (0.9.6 or newer), which you can
find on http://www.twig-project.org.
